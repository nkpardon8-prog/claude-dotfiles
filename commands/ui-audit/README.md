Exhaustive, **report-only** per-tab UI reality audit. For ONE tab/screen of a running web app it enumerates the entire rendered surface across every reachable sub-state into a fail-closed coverage ledger, gives each element a strict verdict (`REAL` / `STATIC-BY-DESIGN` / `FAKE-OR-DEAD` / `UNVERIFIED`, plus a `MODELS-DISAGREE` bucket), and proves reality through three reconciled passes â€” static code trace, live browser x-ray over **raw CDP** (no MCP/Playwright), and screenshot vision â€” with verdict authoring + evidence judgment split ~50/50 Codex/Claude. Output: `findings.json` + `AUDIT.md` + annotated screenshots, handed to `/god-review` or `/implement`. **Never edits application code.**

Transport is **RAW CDP** (rev 3): Node scripts talk to Chrome's `:9222` debug port directly via the global `WebSocket` (zero deps). Screenshots go to files that Claude `Read`s for vision.
